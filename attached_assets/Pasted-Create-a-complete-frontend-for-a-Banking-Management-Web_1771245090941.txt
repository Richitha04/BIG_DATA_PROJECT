Create a complete frontend for a Banking Management Web Application.

Tech Stack:
- React.js (with functional components and hooks)
- Tailwind CSS for styling
- Axios for API communication
- React Router for navigation
- Modern responsive UI suitable for project demonstration

Purpose:
This application will connect to a Node.js + Express + MongoDB backend.
The frontend must allow demonstration of database operations such as:
- Account creation
- Deposit
- Withdrawal
- Fund transfer
- Transaction history
- Viewing database records updates in real-time

Pages Required:

1. Landing Page
   - Bank name and branding
   - Login and Register buttons
   - Clean modern UI

2. Authentication Pages
   - Login
   - Register
   - Form validation
   - JWT token storage (localStorage)

3. Dashboard
   - Welcome user with account details
   - Account balance display
   - Recent transactions preview
   - Buttons for deposit, withdraw, transfer

4. Deposit Page
   - Input amount
   - Submit button
   - Show success message
   - API call to backend

5. Withdraw Page
   - Input amount
   - Validation for insufficient balance
   - API call

6. Transfer Money Page
   - Recipient account number
   - Amount
   - Description
   - Submit button

7. Transaction History Page
   - Table showing:
     - Transaction ID
     - Type
     - Amount
     - Date
     - Status
   - Pagination or scrolling

8. Admin / Database Viewer Page (Important for showcase)
   - Show all users
   - Show all transactions
   - Refresh button
   - This is for demonstrating MongoDB data changes live
